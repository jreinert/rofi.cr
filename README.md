# rofi for crystal

A rofi wrapper for crystal

## Installation

Add it to `shard.yaml`

```yaml
# ...

dependencies:
	rofi:
		github: jreinert/rofi.cr
		version: ~> 0.0.1
# ...
```

## Usage

```crystal
require "rofi"
```
